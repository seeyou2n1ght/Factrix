# Factrix

Factrix 是一个专为个人非专业投资者打造的全景公募基金持仓穿透与量化诊断系统。系统兼具华尔街专业水准与散户极致友好度，将晦涩的金融工程模型转化为通俗易懂的“白话”诊断卡片，帮助您看清底层持仓公司、全方位风险暴露，以及精确的 Alpha/Beta 超额剖析。

## 核心量化引擎与分析维度

系统内置了 8 大核心量化分析引擎，从多个维度深度剖析您的基金组合：

1. **PBSA (Portfolio-Based Style Analysis) 深度持仓穿透引擎**
   - **宏观隔离**：精准剥离股票、债券、商品、现金等四大类宏观资产。
   - **极限下钻**：通过 100% 重归一化算法，将资金精确映射至证监会 19 门类行业，甚至下钻至具体的底层持仓公司（个股）。
   - **防抱团重合度**：自动计算底层持仓个股的交叉重合网络，生成热力图，揭示“表面分散、背地抱团”的隐性集中风险。

2. **CAPM Alpha & Beta 绩效归因引擎 (最新引入)**
   - **超额剖析**：将组合历史收益精准拆解为 **Beta (大盘跟涨被动收益)** 与 **Alpha (基金经理选股主动超额)**。
   - **专业风控比率白话化**：自动测算 夏普比率 (Sharpe)、特雷诺比率 (Treynor)、信息比率 (IR) 以及跟踪误差 (TE)，并通过散点图直观呈现投资性价比。

3. **RBSA (Return-Based Style Analysis) 真实风格回归引擎**
   - 运用 SLSQP 约束优化算法，将基金每日真实净值波动与市场上“大盘价值”、“大盘成长”、“小盘价值”、“小盘成长”、“国债” 5 大纯正标尺比对。不看招募书写了什么，只看基金经理实际买了什么。

4. **Rolling RBSA 风格漂移监测引擎**
   - 通过 60 天滚动时间窗口动态追踪 RBSA 轨迹。当监测到方差超标（经理偷偷追热点）时，自动触发“言行不一”警告。

5. **CVaR (Conditional Value at Risk) 极值压力测试引擎**
   - 采用 Cornish-Fisher 展开式修正尾部非正态分布，模拟计算 95% 与 99% 置信度下的尾部亏损。
   - **真实情景回放**：一键进行 2015年股市踩踏、2022年股债双杀、2020年全球疫情暴跌等压力测试。

6. **人民币真实亏损模拟与风险画像 (最新引入)**
   - 将抽象的统计学概率转化为直观的“人民币预估亏损金额算盘”，精准锚定投资者的真实心理承受阀值。
   - 提供自动匹配的个人投资者风险画像（保守型 / 稳健型 / 积极型）及专属配置建议。

7. **Prospect Theory 前景理论效用引擎**
   - 基于诺贝尔经济学奖 Kahneman-Tversky 损失厌恶模型（λ=2.25），结合 Omega 比率，量化评估当前组合对于非理性个人的“真实心理性价比”。

8. **带摩擦成本的二次规划调仓引擎 (Rebalance Engine)**
   - 智能导航系统不仅指出该买卖哪只基金，还会精准扣除短线赎回惩罚费（如 <7天 1.5% 惩罚）。只有调仓收益绝对覆盖过路费时，才会下达调仓指令。

## 视觉交互与 UI 架构

- **多层交互式 Treemap 漏斗穿透**：通过 Plotly 原生渲染的矩形树图，支持点击色块逐层下钻 `[大类资产 -> 细分行业 -> 基金名称 -> 股票明细]`。自动补偿“隐形重仓及其他 (Others)”色块，保持面积比例严格真实。
- **高定级暗黑美学**：全局采用玻璃拟物态 (Glassmorphism) 卡片、微阴影悬浮渐变及 `plotly_dark` 深色图表体系。
- **白话小贴士体系 (Vernacular Callouts)**：在每个生硬图表旁配置通俗易懂的解读卡片，摒弃学术术语，让非专业投资人秒懂数据背后的真金白银。

## 安装指南

系统基于 Python 3.10+ 开发，推荐使用 `uv` 环境管理器以获得极速体验：

```bash
# 1. 克隆代码仓库
git clone <repository_url>
cd Factrix

# 2. 安装依赖并同步环境
uv sync
```

## 使用说明

1. 导出数据：从“国家清算中心 / 基金E账户 App”直接导出您的原始未删减 `.xlsx` 格式的公募基金持有信息。
2. 放入数据：将该 Excel 文件直接拖入项目根目录。Factrix 的动态路由机制将自动扫描并读取目录内日期最新的数据源（如 `基金E账户App投资者公募基金持有信息-*.xlsx`）。
3. 启动看板：
```bash
uv run streamlit run app.py
```
4. 交互查阅：在浏览器中打开大屏，通过 5 大 Tab 导航深度体检您的私人持仓。

## 安全与隐私声明

- 所有的 API 请求均用于获取公开市场的**净值与行业数据**，绝不会将您的个人账户金额、名称和持仓份额上传至任何云端服务器。
- 请确保本地的 `.gitignore` 包含 `*.xlsx` 等规则，严防在提交代码时将个人真实持仓账本推送至公共代码仓库。
